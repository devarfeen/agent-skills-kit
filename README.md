# Agents Skills Kit

A swiss-army knife collection of reusable skills for AI agents and CLI tools.

## Skills

### Release Notes Skill

A reusable AI agent skill that generates structured release notes and changelogs from development activity.

It supports:

- Git commit history
- Date-based release notes
- Feature summaries
- Development session summaries
- Automatic commit clustering

Designed for AI CLI agents and tools that support installable skills.

#### Example Requests

- Generate release notes for 11 March 2026
- Generate release notes for Partners App on 11 March
- Summarize today's development session
- Write release notes for the improvements
